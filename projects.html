Weather App

HTML | CSS | JavaScript | VS Code | API | GitHub | Bootstrap | Hosting

This project is an interactive weather app created using JavaScript. It provides you with real-life data, since it has an integrated API, provided by OpenWeatherMap. It also enables you to access information such as humidity, wind-speed, precipitation and a five day forecast. All of this information is provided for every city searched. Feel free to try it on my page.

This project was coded by Anna Koval , and is open-sourced.
